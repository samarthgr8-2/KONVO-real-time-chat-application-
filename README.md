# KONVO-real-time-chat-application-
KONVO is a real time full stack chat application using MERN . Used Streamify for real time chat and video calls. Used lucide-react for icons . Daisy UI for various UI components and themes . Used DiceBear forrandom avatar. Tanstack queries for query.  Some UI changes and some code changes are also been made from the reference.
